Class: ScreenController

ScreenController()

new ScreenController()

A ScreenController coordinates a questionnaire, i.e., showing a set of Screens and storing the gathered data. This ScreenController shows the Screens in a predefined order. Applies lifecycle management for the Screens. ATTENTION: `ScreenController.init(parentNode)` must be called before using a ScreenController. Callbacks: ScreenController.callbackScreenFinished() {boolean}: The current screen is done; continue to next screen?
