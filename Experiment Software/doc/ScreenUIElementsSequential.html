Class: ScreenUIElementsSequential

ScreenUIElementsSequential()

new ScreenUIElementsSequential()

A screen that presents one or more UIElements. All UIElements are visible by default. UIElements are enabled one after another, i.e., if its predecessing UIElement reported to be ready the next one is enabled.
