Class: LogMessage

LogMessage()

new LogMessage()

Defines a message that should be logged, consisting of level, location, and the content. The messages _should_ be subdivided in five types according to their relevance: 1. Fatal 2. Error 3. Warn 4. Info 5. Debug DEVELOPER: This class is used internally by LogConsole and should not be accessed directly.
